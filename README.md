📚个人知识库问答系统
项目介绍

一个基于 RAG（检索增强生成）技术的智能问答系统。用户可以上传自己的 PDF 文件（如课程笔记、技术文档、个人简历等），系统会自动构建私有知识库，并基于文档内容回答用户的问题。

核心功能：
1.支持上传多个 PDF 文件，自动解析文档内容
2.使用文本分割技术将长文档切分为适合检索的语义块
3.通过向量嵌入将文档存入向量数据库，实现高效语义检索
4.基于检索到的相关文档，由大模型生成准确、可信的回答
5.提供简洁美观的 Web 交互界面

技术栈

1.Python：主要编程语言
2.LangChain：RAG 流程编排（文档加载、文本分割、检索链、Prompt 模板）
3.Chroma：向量数据库，存储文档嵌入并支持相似度检索
4.Streamlit：Web 界面框架
5.DeepSeek：大语言模型 API（通过硅基流动调用）
6.HuggingFace Embeddings：本地中文嵌入模型（`shibing624/text2vec-base-chinese`）
